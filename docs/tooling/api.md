# API and SDK

Hyli's API allows you to interact with its core components.

## OpenAPI documentation

For partners with access to Hyli infrastructure, API documentation is available on-demand.

## SDK implementations

We currently support Rust and JS environments.

| Language | Repository / Package                                                  |
|----------|-----------------------------------------------------------------------|
| JS       | [hyli on npm](https://www.npmjs.com/package/hyli)                    |
| Rust     | [hyli client sdk on docs.rs](https://crates.io/crates/hyli-client-sdk) |
